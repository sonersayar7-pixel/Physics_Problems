## Section 7: Parametric Trajectory Analysis

**Position:** $\vec{r}(t) = 2t^2\hat{i} + 3t^3\hat{j}$

### Path Equation
Eliminating $t$ yields the relation:
$$y = 3\left(\frac{x}{2}\right)^{1.5} \implies y^2 = \frac{27}{8}x^3$$

### Kinematics Table
| Vector | Expression | Magnitude |
| :--- | :--- | :--- |
| **Velocity** | $4t\hat{i} + 9t^2\hat{j}$ | $t\sqrt{16 + 81t^2}$ |
| **Acceleration** | $4\hat{i} + 18t\hat{j}$ | $\sqrt{16 + 324t^2}$ |

**Conclusion:** The acceleration is **non-constant** because the $\hat{j}$ component is time-dependent.
