## Section 4: Vector Calculus & Kinematics

**Position Vector:**
$$\vec{r}(t) = (3t^2)\hat{i} + (5t - 8t^2)\hat{j}$$

**Velocity Vector:**
$$\vec{v}(t) = \frac{d\vec{r}}{dt} = (6t)\hat{i} + (5 - 16t)\hat{j}$$

**Acceleration Vector:**
$$\vec{a}(t) = \frac{d\vec{v}}{dt} = 6\hat{i} - 16\hat{j}$$

### Analysis Summary
| Property | Value |
| :--- | :--- |
| **Initial Position** ($t=0$) | $(0, 0)$ |
| **Initial Velocity** ($t=0$) | $5\hat{j} \text{ m/s}$ |
| **Constant Acceleration** | $6\hat{i} - 16\hat{j} \text{ m/s}^2$ |
| **Motion Type** | Uniformly Accelerated 2D Motion |
