## 7. Standing Wave Modes: Higher Harmonics

When a string fixed at both ends vibrates, it can do so in several different patterns called **harmonics** or **standing wave modes**. Rather than just one large arc (the fundamental frequency), the string can form multiple vibrating loops. The points of maximum movement in these loops are called **antinodes**. 

The harmonic number ($n$) directly corresponds to the number of antinodes (or loops) present on the string.

### Step 1: The Harmonic Wavelength Formula
For any given string of length $L$, the wavelength of the $n$-th harmonic can be calculated using a simple relationship. Because each loop (antinode) represents exactly half of a wavelength, we can use the following formula:
$$\lambda_n = \frac{2L}{n}$$

**Where:**
* **$\lambda_n$:** The wavelength of the $n$-th harmonic.
* **$L$:** The total physical length of the string.
* **$n$:** The integer representing the harmonic mode (and the number of antinodes).

---

### Step 2: The Scenario Variables
Let's analyze a specific scenario where a string is vibrating in its 4th harmonic. 
* **String length ($L$):** $80 \text{ cm}$
* **Harmonic number/Antinodes ($n$):** $4$

---

### Step 3: Calculating the Wavelength
Substitute our known variables into the general harmonic formula:
$$\lambda = \frac{2 \times 80 \text{ cm}}{4}$$

First, multiply the length of the string by two:
$$\lambda = \frac{160 \text{ cm}}{4}$$

Next, divide by the number of antinodes:
$$\lambda = 40 \text{ cm}$$

---

### Step 4: Converting to Standard Units
In physics, it is always best practice to convert centimeters to the standard SI unit of meters. Since there are 100 centimeters in a meter, we divide by 100:
$$\lambda = 0.4 \text{ m}$$

### Conclusion for Presentation
When an **80 cm** string vibrates in its 4th harmonic mode ($n=4$), it forms four distinct loops. The formula proves that the physical wavelength of the wave traveling along this string is exactly **40 cm** (or **0.4 m**). Since a full wave consists of two loops, this makes perfect physical sense—two loops take up 40 cm, meaning all four loops fit perfectly within the 80 cm string!
